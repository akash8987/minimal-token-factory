# Minimal Token Factory

An expert-grade, flat-structured repository for deploying customized ERC-20 tokens on EVM-compatible chains (Ethereum, Polygon, Arbitrum, etc.).

## Features
* **Standard Compliance:** Uses OpenZeppelin v5.0 primitives.
* **Capped Supply:** Prevents inflation beyond a predefined limit.
* **Burnable:** Built-in mechanism for deflationary tokenomics.
* **Gas Optimized:** Clean Solidity implementation for lower deployment costs.

## Quick Start
1. Install dependencies: `npm install`
2. Compile contracts: `npx hardhat compile`
3. Deploy: `npx hardhat run deploy.js`

## Security
This code is for educational purposes. Ensure you conduct a professional audit before deploying to Ethereum Mainnet.
